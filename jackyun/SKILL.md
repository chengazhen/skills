# 吉客云用户操作指南 Skill

## 使用方式

本技能包含吉客云 ERP 系统的完整用户手册（837 篇文档，27 个业务模块）。

**关键原则：按需加载** — 不要一次性读取所有参考文件。根据用户问题所属的模块，只加载对应的 reference 文件。

## 工作流程

1. **识别模块** — 根据用户问题判断属于哪个业务模块
2. **加载参考** — 用 `read` 工具加载对应的 `references/XX.md` 文件
3. **定位内容** — 在文件中搜索相关的 `## 功能名称` 章节
4. **回答问题** — 基于文档内容给出操作步骤和说明

## 回答规范

- 引用文档时说明来源模块
- 操作步骤用有序列表呈现
- 涉及的功能路径用 `菜单 > 子菜单 > 功能` 格式
- 如文档中无相关内容，坦诚告知并建议联系吉客云客服

## 注意事项

- 文档基于吉客云 2020 版本，部分功能可能已更新
- 涉及 API 对接的文档需要额外验证接口是否仍有效
- 文档中图片链接已失效，仅保留文字描述
